Une application rigolote pour simplement jouer un son quand
vous appuyez sur un bouton, ou quand vous bougez le téléphone.
Ceci est ma première app Android, mais c'est marant à utiliser.
Vous pouvez virtuellement foueter vos amis, ou vous moquez d'eux
(mais pas trop).
Vous pouvez jouer le Dun Dun Duuun ou le son du Suspens lors
d'une murder party par exemple :-) !
Vous avez vous les épisodes de The Big Bang Theory quand ils se fouettent
les uns les autres ou qu'il font la murder party ? Les sons son presque
les mêmes, et vous pouvez utiliser l'appli comme ils le font dans la série.

Aide :
Appuyez sur un bouton, et le son est joué. C'est facile :-) !
Le boutton avec l'icône rejoue le dernier son.
Vous pouvez utiliser un clavier (touche 1 à 5 et R pour rejouer).
Le dernier son est aussi joué quand vous faites un mouvement de fouet avec le téléphone.

Les sons ne peuvent pas être changés (cela viendra dans une future version).
